# Places You've Been

#### By Jeff Seymour, Greg Kanteres

## Description
A program to enter places you've been.  Exploration with methods in Object-Oriented Ruby programming.

## Technologies Used
HTML, CSS, Ruby, RSpec, Capybara, Sinatra

## Setup
Clone repository  
```bundle install```  
```ruby app.rb```  
View in browser at localhost:4567
